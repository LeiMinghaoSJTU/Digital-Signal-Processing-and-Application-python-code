# 齐次差分方程通解求解方法

## 齐次差分方程的一般形式

对于一个 $k$ 阶线性常系数齐次差分方程：

$$a_0y_n + a_1y_{n-1} + a_2y_{n-2} + \cdots + a_ky_{n-k} = 0$$

其中 $a_0, a_1, \ldots, a_k$ 是常数，且 $a_0 \neq 0$。

## 求解步骤

### 第一步：写出特征方程

将 $y_n$ 替换为 $r^n$，得到特征方程：

$$a_0r^k + a_1r^{k-1} + a_2r^{k-2} + \cdots + a_k = 0$$

### 第二步：求解特征根

解上述特征方程，得到 $k$ 个特征根 $r_1, r_2, \ldots, r_k$（可能有重根或复根）。

### 第三步：根据特征根类型写出通解

根据特征根的不同情况，通解形式如下：

#### 情况一：所有特征根为互异实根

通解形式为：
$$y_n = C_1r_1^n + C_2r_2^n + \cdots + C_kr_k^n$$

#### 情况二：存在重实根

若 $r$ 是 $m$ 重实根，则对应的解部分为：
$$(C_1 + C_2n + C_3n^2 + \cdots + C_mn^{m-1})r^n$$

#### 情况三：存在共轭复根

若有一对共轭复根 $r = α ± βi$，可将其转换为极坐标形式：
$$r = ρ(\cos θ ± i\sin θ)$$
其中 $ρ = \sqrt{α^2 + β^2}$，$θ = \arctan(β/α)$

对应的解部分为：
$$ρ^n[C_1\cos(nθ) + C_2\sin(nθ)]$$

### 第四步：组合通解

将不同特征根对应的解部分组合起来，得到完整的通解。

## 示例

求解二阶齐次差分方程：$y_n - 5y_{n-1} + 6y_{n-2} = 0$

1. 特征方程：$r^2 - 5r + 6 = 0$
2. 求解特征根：$(r-2)(r-3)=0$，得 $r_1=2$, $r_2=3$
3. 通解：$y_n = C_1 \cdot 2^n + C_2 \cdot 3^n$

## 记忆技巧

1. **特征方程**：将 $y_{n-i}$ 替换为 $r^{k-i}$（$k$ 为阶数）
2. **实根**：$r^n$ 形式
3. **重根**：乘以 $n$ 的多项式
4. **复根**：转换为三角函数形式，注意有 $ρ^n$ 因子
