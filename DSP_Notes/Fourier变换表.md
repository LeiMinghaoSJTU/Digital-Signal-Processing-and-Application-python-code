好的，我们来介绍一些**最常用函数**的**傅里叶变换对**。傅里叶变换将时域（或空域）函数 $f(t)$ 映射到频域函数 $F(ω)$，揭示了信号包含的频率成分及其强度。

傅里叶变换（FT）及其逆变换（IFT）的标准定义如下：

*   **傅里叶变换 (FT):** $F(ω) = ∫_{-∞}^{∞} f(t) e^{-jωt} dt$
*   **逆傅里叶变换 (IFT):** $f(t) = (1/2π) ∫_{-∞}^{∞} F(ω) e^{jωt} dω$

**注意：** 傅里叶变换的定义有多种约定（例如，$2π$ 因子的位置、指数项的符号）。上述定义是工程和物理学中最常用的一种（有时称为“非酉角频率”形式）。以下变换对均基于此定义。

---

### 常用函数的傅里叶变换对

1.  **单位冲激函数 (Dirac Delta Function) - δ(t)**
    *   **时域：** $f(t) = δ(t)$
    *   **频域：** $F(ω) = 1$
    *   **物理意义：** 一个理想化的瞬时无限大脉冲（能量集中在 $t=0$）。它的频谱在所有频率上都是常数 $1$，表示它包含了**所有频率成分**，且所有频率分量的**幅度相同**（无限宽频谱）。这是傅里叶变换中最基础、最重要的变换对之一。

2.  **常数函数 (Constant Function)**
    *   **时域：** $f(t) = 1$ (或常数 $A$)
    *   **频域：** $F(ω) = 2π δ(ω)$
    *   **物理意义：** 一个不随时间变化的直流（DC）信号。它的频谱仅在**零频率（ω=0）** 处有一个冲激，表示它只包含**直流分量**，没有任何交流频率成分。$2π$ 因子源于使用的傅里叶变换定义约定。

3.  **复指数函数 (Complex Exponential Function)**
    *   **时域：** $f(t) = e^{jω₀t}$
    *   **频域：** $F(ω) = 2π δ(ω - ω₀)$
    *   **物理意义：** 一个频率为 $ω₀$ 的纯单频振荡。它的频谱是在频率 $ω = ω₀$ 处的一个冲激，表示信号的所有能量都集中在**这一个频率** $ω₀$ 上。

4.  **余弦函数 (Cosine Function)**
    *   **时域：** $f(t) = cos(ω₀t)$
    *   **频域：** $F(ω) = π [δ(ω - ω₀) + δ(ω + ω₀)]$
    *   **物理意义：** 频率为 $ω₀$ 的实余弦波。根据欧拉公式 $cos(ω₀t) = (e^{jω₀t} + e^{-jω₀t})/2$，它的频谱由两个冲激组成，分别位于**正频率** $ω = ω₀$ 和**负频率** $ω = -ω₀$ 处，且幅度都是 $π$。负频率是数学形式的结果，与正频率一起表示一个单一的实频率分量。

5.  **正弦函数 (Sine Function)**
    *   **时域：** $f(t) = sin(ω₀t)$
    *   **频域：** $F(ω) = jπ [δ(ω + ω₀) - δ(ω - ω₀)]$
    *   **物理意义：** 频率为 $ω₀$ 的实正弦波。根据欧拉公式 $sin(ω₀t) = (e^{jω₀t} - e^{-jω₀t})/(2j)$，它的频谱也由两个冲激组成，分别位于 $ω = ω₀$ 和 $ω = -ω₀$。与余弦不同的是，这两个冲激的相位相差 $180°$（体现在 $j$ 和 $-$ 号上）。

6.  **矩形脉冲/门函数 (Rectangular Pulse / Gate Function)**
    *   **时域：** $f(t) = rect(t/τ) = { 1, |t| < τ/2; 0, |t| > τ/2 }$
    *   **频域：** $F(ω) = τ * sinc(ωτ/2)$
        *   其中 $sinc(x) = sin(x)/x$ (有时归一化的 $sinc(πx)$ 定义更常见，但这里用 $sin(x)/x$ 形式)。
    *   **物理意义：** 一个宽度为 $τ$、高度为 $1$ 的矩形脉冲。它的频谱是 $sinc$ 函数。$sinc$ 函数的主瓣宽度与 $1/τ$ 成正比（脉冲越宽，频谱越窄），旁瓣幅度逐渐衰减。零点出现在 $ω = ±2kπ/τ$ ($k=1,2,3,...$)。这个变换对在信号采样和滤波中极其重要。

7.  **单边指数衰减 (Causal Exponential Decay)**
    *   **时域：** $f(t) = e^{-at} u(t)$ ($a > 0$, $u(t)$ 是单位阶跃函数)
    *   **频域：** $F(ω) = 1/(a + jω)$
    *   **物理意义：** 从 $t=0$ 开始按指数规律衰减的信号（常见于 RC 电路响应、阻尼振荡）。它的频谱是一个复值函数，幅度谱 $|F(ω)| = 1/√(a² + ω²)$ 随频率增加而衰减，相位谱 $∠F(ω) = -arctan(ω/a)$。$a$ 越大（衰减越快），频谱越宽；$a$ 越小（衰减越慢），频谱越窄。

8.  **高斯函数 (Gaussian Function)**
    *   **时域：** $f(t) = e^{-a t²}$ ($a > 0$)
    *   **频域：** $F(ω) = √(π/a) e^{-ω²/(4a)}$
    *   **物理意义：** 钟形曲线。它的傅里叶变换**仍然是高斯函数**（形状相同，只是参数变化）。这是一个非常重要的性质，意味着高斯脉冲在时域和频域都具有良好的局部性（能量集中），是使时频不确定性乘积达到最小的函数。

9.  **符号函数 (Signum Function)**
    *   **时域：** $f(t) = sgn(t) = { -1, t < 0; 0, t=0; 1, t>0 }$
    *   **频域：** $F(ω) = 2/(jω)$
    *   **物理意义：** 表示信号的正负极性。它在 $ω=0$ 处有奇点（无穷大），反映了其直流分量为零 ($f(t)$ 是奇函数) 且包含所有频率分量（低频分量尤其显著）。其频谱是纯虚奇函数。

10. **单位阶跃函数 (Unit Step Function)**
    *   **时域：** $f(t) = u(t) = { 0, t < 0; 1, t > 0 }$ (在 $t=0$ 处通常定义为 $1/2$ 或不定义)
    *   **频域：** $F(ω) = π δ(ω) + 1/(jω)$
    *   **物理意义：** 表示信号的突然开始或开关动作。它的频谱由两部分组成：
        *   $π δ(ω)$：代表其平均值或直流分量 ($1/2$)。
        *   $1/(jω)$：代表其交流分量（符号函数 $sgn(t)$ 的频谱是 $2/(jω)$，而 $u(t) = 1/2 + (1/2)sgn(t)$）。它在 $ω=0$ 处也有奇点。

---

### 总结

*   **δ(t) ↔ 1**: 瞬时脉冲包含所有频率。
*   **1 ↔ 2π δ(ω)**: 直流信号只有零频率分量。
*   **e^{jω₀t} ↔ 2π δ(ω - ω₀)**: 复单频振荡在频域是一个冲激。
*   **cos(ω₀t) ↔ π [δ(ω-ω₀) + δ(ω+ω₀)]**: 实余弦波在 ±ω₀ 处有冲激。
*   **sin(ω₀t) ↔ jπ [δ(ω+ω₀) - δ(ω-ω₀)]**: 实正弦波在 ±ω₀ 处有冲激（带相位差）。
*   **rect(t/τ) ↔ τ sinc(ωτ/2)**: 矩形脉冲的频谱是 sinc 函数（主瓣宽度 ∝ 1/τ）。
*   **e^{-at}u(t) (a>0) ↔ 1/(a + jω)**: 单边指数衰减的频谱随频率增加而衰减。
*   **e^{-a t²} ↔ √(π/a) e^{-ω²/(4a)}**: 高斯函数的变换还是高斯函数（最优时频局部性）。
*   **sgn(t) ↔ 2/(jω)**: 符号函数的频谱在零频处奇异，随 1/ω 衰减（纯虚奇）。
*   **u(t) ↔ π δ(ω) + 1/(jω)**: 阶跃函数包含直流分量 (δ(ω)) 和交流分量 (1/(jω))。

这些变换对是信号与系统分析、通信、图像处理等领域的基础工具。理解和记忆它们对于分析信号的频率特性、设计滤波器、理解采样定理等都至关重要。在应用时，务必注意所使用的傅里叶变换定义约定（特别是 $2π$ 因子的位置）。
