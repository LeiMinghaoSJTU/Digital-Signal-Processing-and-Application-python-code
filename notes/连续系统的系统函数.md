连续时间系统的**系统函数 \(H(s)\)**（也称为传递函数或网络函数）是描述线性时不变（LTI）系统输入-输出关系的**复频域（s域）模型**。其定义基于拉普拉斯变换，核心思想是将时域的微分方程转换为s域的代数方程，从而简化系统分析。

### 定义步骤
1. **系统建模（时域微分方程）**  
   一个n阶连续LTI系统的动态行为通常由常系数线性微分方程描述：
   \[
   a_n \frac{d^n y(t)}{dt^n} + a_{n-1} \frac{d^{n-1} y(t)}{dt^{n-1}} + \cdots + a_0 y(t) = b_m \frac{d^m x(t)}{dt^m} + \cdots + b_0 x(t)
   \]
   其中：
   - \(x(t)\) 是输入信号，
   - \(y(t)\) 是输出信号，
   - \(a_i, b_j\) 是系统参数（常数）。

2. **应用拉普拉斯变换**  
   对微分方程两边进行**拉普拉斯变换**，利用其性质（如线性性、微分性质）：
   - 若 \(\mathcal{L}\{y(t)\} = Y(s)\)，则 \(\mathcal{L}\left\{\frac{d^k y(t)}{dt^k}\right\} = s^k Y(s) - \sum_{i=0}^{k-1} s^{k-1-i} y^{(i)}(0^-)\)  
     （假设初始条件为零：\(y^{(i)}(0^-) = 0\)）

3. **零初始条件假设**  
   在系统函数定义中，通常假设**系统初始松弛**（所有初始条件为零）。此时微分方程转换为：
   \[
   \left( a_n s^n + a_{n-1} s^{n-1} + \cdots + a_0 \right) Y(s) = \left( b_m s^m + \cdots + b_0 \right) X(s)
   \]
   其中 \(X(s) = \mathcal{L}\{x(t)\}\)，\(Y(s) = \mathcal{L}\{y(t)\}\).

4. **定义系统函数 \(H(s)\)**  
   输出拉普拉斯变换 \(Y(s)\) 与输入拉普拉斯变换 \(X(s)\) 的比值即为系统函数：
   \[
   \boxed{H(s) = \frac{Y(s)}{X(s)} = \frac{b_m s^m + b_{m-1} s^{m-1} + \cdots + b_0}{a_n s^n + a_{n-1} s^{n-1} + \cdots + a_0}}
   \]

---

### 关键特性
1. **有理函数形式**  
   \(H(s)\) 是复变量 \(s = \sigma + j\omega\) 的**有理分式**（分子分母均为s的多项式）。

2. **极点与零点**  
   - **零点**：分子多项式根（使 \(H(s)=0\) 的s值）  
   - **极点**：分母多项式根（使 \(H(s) \to \infty\) 的s值）  
   *极点位置决定系统稳定性（所有极点需在s左半平面）*。

3. **物理意义**  
   - **冲激响应的拉普拉斯变换**：  
     \[
     H(s) = \mathcal{L}\{h(t)\}, \quad h(t) = \text{系统冲激响应}
     \]
   - **频率响应的推广**：  
     令 \(s = j\omega\)，则 \(H(j\omega)\) 即系统的**频率响应**（含幅频/相频特性）。

4. **系统行为表征**  
   | 特性         | s域表现                           |
   | ------------ | --------------------------------- |
   | **稳定性**   | 所有极点 \(\text{Re}(s) < 0\)     |
   | **因果性**   | 分子阶数 \(m \leq n\)（分母阶数） |
   | **滤波特性** | 零极点分布决定频率选择性          |

---

### 示例：RC低通滤波器的系统函数
- **时域方程**：  
  \[
  RC \frac{dy(t)}{dt} + y(t) = x(t) \quad (x(t): \text{输入电压},  y(t): \text{输出电压})
  \]
- **拉普拉斯变换（零初始条件）**：  
  \[
  RC \cdot s Y(s) + Y(s) = X(s) \implies (RCs + 1)Y(s) = X(s)
  \]
- **系统函数**：  
  \[
  H(s) = \frac{Y(s)}{X(s)} = \frac{1}{RCs + 1} = \frac{1/RC}{s + 1/RC}
  \]
  - **极点**：\(s = -1/RC\)（在左半平面，系统稳定）  
  - **频率响应**：\(H(j\omega) = \frac{1}{j\omega RC + 1}\) → 低通特性  

---

### 为什么需要 \(H(s)\)？
1. **简化分析**：将微分运算转化为代数运算。
2. **稳定性判据**：通过极点位置快速判断稳定性。
3. **频率响应设计**：通过零极点配置设计滤波器（如Butterworth、Chebyshev）。
4. **系统互联**：串联、并联、反馈系统的总 \(H(s)\) 易通过公式计算（如 \(H_{\text{串}}(s) = H_1(s)H_2(s)\)）。

> 总结：**\(H(s)\) 是连续LTI系统在复频域的核心描述工具**，将时域微分方程转化为s域的有理函数，通过零极点分布揭示系统的稳定性、频率响应及动态特性。
