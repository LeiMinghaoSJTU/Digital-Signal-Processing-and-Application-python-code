## 快速开始

### 环境要求

```bash
pip install numpy matplotlib scipy
```

### 运行示例

每个Python文件都包含 `demo()` 函数，可以直接运行查看效果：

```bash
# 运行离散信号与卷积演示
python discrete_signal_and_convolution.py

# 运行FFT演示
python dft_and_fft.py

# 运行滤波器设计演示
python fir_filter_window_method.py
```

### 在代码中导入使用

```python
from discrete_signal_and_convolution import discrete_convolution, unit_impulse
from dft_and_fft import fft_iterative, dft_direct
from fir_filter_window_method import fir_lowpass_window

# 使用示例
import numpy as np
x = np.array([1, 2, 3, 4])
h = np.array([1, 1, 1])
y = discrete_convolution(x, h)
```

## 代码功能详细说明

### 1. 离散信号与卷积 (`discrete_signal_and_convolution.py`)

- **单位脉冲序列**: δ[n-n₀] 的生成
- **单位阶跃序列**: u[n-n₀] 的生成
- **指数序列**: aⁿ 的生成
- **正弦序列**: A·sin(ωn+φ) 的生成
- **离散卷积**: 直接计算方法，时间复杂度 O(N²)
- **差分方程求解**: 使用 `scipy.signal.lfilter`
- **系统稳定性判断**: 通过极点位置判断

### 2. z变换 (`z_transform_and_inverse.py`)

- **z变换计算**: 多项式表示
- **逆z变换**: 部分分式展开法
- **系统函数分析**: 零极点分析、稳定性判断
- **频率响应**: 幅频响应和相频响应

### 3. DFT与FFT (`dft_and_fft.py`)

- **直接DFT计算**: O(N²) 复杂度
- **DIT-FFT**: 按时间抽取的基2-FFT
- **DIF-FFT**: 按频率抽取的基2-FFT
- **迭代FFT**: 非递归实现
- **线性卷积**: 基于DFT的快速卷积

### 4. FIR滤波器设计 (`fir_filter_window_method.py`)

- **窗函数**: 矩形窗、汉宁窗、汉明窗、布莱克曼窗、凯泽窗
- **滤波器类型**: 低通、高通、带通、带阻
- **窗函数特性比较**: 主瓣宽度、旁瓣衰减

### 5. IIR滤波器设计 (`iir_filter_design.py`)

- **设计方法**: 冲激响应不变法、双线性变换法
- **滤波器类型**: 巴特沃斯、切比雪夫I/II型、椭圆滤波器
- **频率变换**: 低通→高通/带通/带阻

### 6. DCT变换 (`dct_transform.py`)

- **DCT类型**: DCT-I、DCT-II、DCT-III、DCT-IV
- **快速算法**: 基于FFT的快速DCT
- **二维DCT**: 图像压缩应用
- **能量压缩特性**: 信号压缩演示

### 7. 功率谱估计 (`power_spectrum_estimation.py`)

- **经典方法**: 周期图法、相关图法
- **改进方法**: Welch方法
- **参数模型**: AR模型、Yule-Walker方程、Burg算法

### 8. 维纳与卡尔曼滤波 (`wiener_filter.py`)

- **FIR维纳滤波**: 最优线性滤波
- **维纳反卷积**: 图像/信号恢复
- **卡尔曼滤波**: 状态估计、最优预测
- **自适应LMS**: 在线参数估计

### 9. Wigner分布 (`wigner_distribution.py`)

- **Wigner-Ville分布**: 高分辨率时频分析
- **伪WVD**: 加窗减少交叉项
- **SPWVD**: 平滑伪Wigner分布
- **与STFT比较**: 分辨率和交叉项的权衡

### 10. 采样定理 (`sampling_theorem.py`)

- **采样过程**: 连续信号离散化
- **奈奎斯特定理**: 采样频率 ≥ 2×最高频率
- **信号重构**: 理想sinc插值、ZOH、线性插值
- **抗混叠滤波**: 采样前低通滤波
- **A/D量化**: 不同位数的量化效果
